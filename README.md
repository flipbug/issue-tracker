# Web 3 - Issue Tracker
*Author: Daniel Milenkovic (milendan)*

Simple issue tracker project created as part of the web3 lectures. 
Build with node.

## Setup 

The project is build with npm/gulp:
```shell
// install dependencies
npm install

// build the project 
gulp

// start the npm server
npm start
```
